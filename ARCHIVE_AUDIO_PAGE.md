# Archive - Ancienne page casque audio

Ce fichier trace l'archivage de l'ancien livrable hors contexte (vente d'un casque audio physique).

## Contexte d'archivage
L'ancien livrable contenait:

- vente d'un objet physique (casque audio),
- logique de panier et ajout produit,
- prix unitaires type e-commerce.

Ces éléments ont été supprimés du flux principal pour aligner le projet sur le contexte SaaS demandé.

## Fichiers supprimés (legacy e-commerce)

- `src/components/Header.tsx`
- `src/components/Hero.tsx`
- `src/components/ProductCard.tsx`
- `src/components/Toast.tsx`
- `src/data.ts`

## Remplacement
Le nouveau flux principal est dans:

- `src/App.tsx`

Il présente une page produit SaaS orientée automatisation IA e-commerce + social media, avec une offre de **2 mois gratuits d'abonnement**.
